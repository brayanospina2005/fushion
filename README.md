# Fushion - Plataforma de Servicios Empresariales

![Logo](src/assets/icon.svg)

## 🚀 Descripción

Fushion es una plataforma web moderna desarrollada con React y TypeScript que ofrece servicios empresariales integrales. Diseñada para proporcionar una experiencia de usuario excepcional y una interfaz intuitiva, la plataforma facilita la conexión entre empresas y servicios profesionales.

## ✨ Características

- 🌐 Interfaz multilingüe (Español/Inglés)
- 🎨 Diseño moderno y responsive
- ⚡ Rendimiento optimizado
- 🛠️ Arquitectura modular y escalable
- 🔒 Prácticas de seguridad implementadas
- 📱 Experiencia móvil optimizada

## 🛠️ Tecnologías

- **Frontend:**
  - React 18
  - TypeScript
  - Tailwind CSS
  - Vite

- **Herramientas de Desarrollo:**
  - ESLint
  - PostCSS
  - Git

## 📦 Instalación

1. Clona el repositorio:
```bash
git clone https://github.com/brayanospina2005/fushion.git
```

2. Instala las dependencias:
```bash
cd fushion
npm install
```

3. Inicia el servidor de desarrollo:
```bash
npm run dev
```

## 🔧 Scripts Disponibles

- `npm run dev` - Inicia el servidor de desarrollo
- `npm run build` - Construye la aplicación para producción
- `npm run lint` - Ejecuta el linter
- `npm run preview` - Vista previa de la build de producción
